# 使用的项目地址

[siteproxy](https://github.com/netptop/siteproxy/)

# 使用方法

## 改成自己的网页

绿色 Code -> Download ZIP

下载所有文件，编辑 content.js 文件

```
const customUrl = "https://xxx.com/xxx/";
```

## 安装

浏览器扩展页面-打勾开发人员模式、允许安装其他应用商店的扩展

加载解压缩的扩展（整个文件夹）

## 使用

网页地址栏右边可以将扩展在工具栏中显示

点击使用web代理打开/点击跳出代理

适用场景：将代理的搜索引擎设置为默认搜索引擎之后，国外与国内网站总有一个不方便打开，点击扩展快速使用代理打开/不使用打开。
